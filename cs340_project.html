CS-340 Project: Web Application

Project Overview

This project involved creating a web application with a database backend using Python and MongoDB. The application allows users to perform CRUD operations on animal shelter data.

Role & Responsibilities

In this project, I was responsible for designing the database schema, implementing the backend logic, and integrating the frontend using Dash for data visualization.

Skills Applied

  • MongoDB Database Management
  • Data Visualization with Dash
